Beasts + Heroes

Web Design & Web Development

Client:

Emortal Sports Inc.


Goal:

Create the look and feel of the website


Role:

Front-End Developer and Ui Designer

Beasts and Heroes is an upcoming Virtual Reality (VR) game for Seattle’s startup Emortal Sports.

Collaborated with the game company Emortal Sports, Inc. to build and develop the game’s web presence, and document the experience of Beasts and Heroes.

Was personally responsible of creating the visual design and HTML5/CSS3 templating for a virtual reality game website where people will be able to learn, discover and presence the world of BnH.
